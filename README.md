## Mind Meetme 

### Source Backend (User)

### How to run this project?

#### 1. Clone project

```bash
$ git clone https://github.com/levanthuan2001qttl/Mind_backend
```

#### 2. Change directory to folder server

```bash
$ cd Mind_backend
```

#### 3. Install dependencies

```bash
$ npm install
```

#### 4. Run in developement mode

```bash
$ npm start
```
